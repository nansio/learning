### 响应式图片

```css
.img{
  width: 100%;
  hegiht: auto
}
```

### 图片滤镜

```css
.img{
  filter: grayscale(100%);
}
```
可选值：

- `filter: blur(4px);`
- `filter: brightness(200%);`
- `filter: contrast(150%);`
- `filter: grayscale(100%);`
- `filter: hue-rotate(180deg);`
- `filter: invert(100%);`
- `filter: opacity(50%);`
- `filter: saturate(7);`
- `filter: sepia(100%);`
- `filter: drop-shadow(8px 8px 10px green);`
